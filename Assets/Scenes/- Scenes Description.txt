씬 설명하기 위한 글입니다.

일단 폴더가 3개가 있을겁니다.
각자 닉네임으로 제가 변경해드릴테오니 
앞으로 뭔가 씬을 새로 파서 작업이 필요하신 경우
해당 닉네임으로 된 폴더 내에서 마음껏 생성하신 후 
작업하시면 감사드리겠습니다.

============================
Start Scene은 게임 시작씬입니다.

게임 파일 exe를 시작하였을 때 보여질 씬으로

[          게임 시작,         게임 종료         ]

버튼을 선택할 수 있습니다. 
유저들이 게임 시작 버튼을 누르면 
MainScene으로 씬 전환을 하고,
종료버튼을 누를 경우 프로그램을 종료하도록
할 것입니다.


=============================
Main Scene은 게임 진행 씬입니다.

게임이 본격적으로 이루어지는 씬이라고 보시면 됩니다.
